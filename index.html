រឿងរ៉ាវរបស់ខ្ញុំ

ខ្ញុំបានចាប់ផ្តើមដំណើររបស់ខ្ញុំដោយការចង់ដឹងចង់ឃើញ។ ភាពឯកោបានបង្រៀនខ្ញុំពីភាពរឹងមាំ ហើយការលំបាកបានបង្រៀនខ្ញុំពីប្រាជ្ញា។ រាល់បន្ទាត់កូដដែលខ្ញុំសរសេរ គឺជារឿងរ៉ាវនៃការលូតលាស់របស់ខ្ញុំ។

អានបន្តទៀត

ជំនាញរបស់ខ្ញុំ

HTML

ខ្ញុំអាចបង្កើតរចនាសម្ព័ន្ធគេហទំព័រដែលរឹងមាំ និងមានអត្ថន័យ (semantic) ដើម្បីធានាបាននូវភាពងាយស្រួលក្នុងការចូលប្រើ (accessibility) និង SEO ល្អ។

CSS

ជំនាញក្នុងការបង្កើត Layout ដែលអាចបត់បែនបាន (responsive) ជាមួយ Flexbox និង Grid, និងបង្កើតចលនា (animations) ដ៏រស់រវើក។

JavaScript

ខ្ញុំអាចធ្វើឱ្យគេហទំព័រមានអន្តរកម្ម, គ្រប់គ្រងព្រឹត្តិការណ៍ (events), ធ្វើការជាមួយ DOM និងតភ្ជាប់ទៅកាន់ APIs។

ដំណើរជីវិត

2023 - ចំណុចចាប់ផ្តើម

ចាប់ផ្តើមស្វែងយល់ពីពិភពកូដដំបូង ដោយរៀនពី HTML និង CSS។ បានបង្កើតគេហទំព័រដំបូងបំផុតដែលជា Static page ដ៏សាមញ្ញ។

2024 - ការឈានចូល JavaScript

បានចាប់ផ្តើមរៀន JavaScript ដើម្បីធ្វើឱ្យគេហទំព័រមានអន្តរកម្ម។ បានបង្កើតគម្រោងតូចៗដូចជា Calculator និង To-Do List។

ដើមឆ្នាំ 2025 - ការស្វែងយល់ពី Backend

ចាប់ផ្តើមរៀនអំពី APIs និងបានបង្កើត Telegram Bot ដំបូង (Dictionary Bot) ដោយប្រើប្រាស់ Serverless Functions។

បច្ចុប្បន្ន 2025 - ការកសាង Portfolio

កំពុងផ្តោតសំខាន់លើការកសាង Portfolio នេះឡើង, ដោយប្រើប្រាស់នូវរាល់ជំនាញដែលបានរៀន និងកែលម្អវាជារៀងរាល់ថ្ងៃ។
