រឿងរ៉ាវរបស់ខ្ញុំ

ខ្ញុំបានកើតនៅក្នុងពិភពដ៏ស្ងប់ស្ងាត់មួយ — ជាកន្លែងដែលខ្ញុំត្រូវធំដឹងក្តីដោយគ្មានការគាំទ្រ គ្មានការលើកទឹកចិត្ត និងគ្មាននរណាម្នាក់សួរថា "តើអ្នកសុខសប្បាយទេ?"

តាំងពីនៅក្មេង ខ្ញុំបានក្លាយជានិមិត្តរូបនៃការរស់រានមានជីវិត។ ខ្ញុំមិនដែលរំពឹងថានឹងមានជំនួយទេ។ ខ្ញុំបានក្លាយជាពន្លឺផ្ទាល់ខ្លួនរបស់ខ្ញុំនៅក្នុងភាពងងឹត។ ភាពឯកោបានបង្រៀនខ្ញុំពីភាពរឹងមាំ ហើយការលំបាកបានបង្រៀនខ្ញុំពីប្រាជ្ញា។

ខ្ញុំមិនមានអនាគតដ៏ល្អឥតខ្ចោះដែលបានគូសវាសទុកជាមុនទេ។ ប៉ុន្តែខ្ញុំមានក្តីសុបិនមួយ គឺដើម្បីក្លាយជាមនុស្សល្អជាងម្សិលមិញ។ ដើម្បីប្រើប្រាស់កូដ ការច្នៃប្រឌិត និងភាពក្លាហាន ដើម្បីកសាងជីវិតប្រកបដោយអត្ថន័យ — ម្ដងមួយបន្ទាត់។

អានបន្តទៀតឬ?

ជំនាញរបស់ខ្ញុំ

HTML

ខ្ញុំអាចបង្កើតរចនាសម្ព័ន្ធគេហទំព័រដែលរឹងមាំ និងមានអត្ថន័យ (semantic) ដើម្បីធានាបាននូវភាពងាយស្រួលក្នុងការចូលប្រើ (accessibility) និង SEO ល្អ។

CSS

ជំនាញក្នុងការបង្កើត Layout ដែលអាចបត់បែនបាន (responsive) ជាមួយ Flexbox និង Grid, និងបង្កើតចលនា (animations) ដ៏រស់រវើកដើម្បីឱ្យអ្នកប្រើប្រាស់មានបទពិសោធន៍ល្អ។

JavaScript

ខ្ញុំអាចធ្វើឱ្យគេហទំព័រមានអន្តរកម្ម, គ្រប់គ្រងព្រឹត្តិការណ៍ (events), ធ្វើការជាមួយ DOM, និងតភ្ជាប់ទៅកាន់ APIs ដើម្បីទាញយកទិន្នន័យពីខាងក្រៅ។
